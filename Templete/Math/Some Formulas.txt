f(n) = a*f(n-1) + b*f(n-2) +c;

a b 1     f(n-1)        f(n)
1 0 0  *  f(n-2)    =   f(n-1)
0 0 1     c             c

----------------------------------

f(n) = c^n - f(n-1)     c为常数

-1 c    *   f(n-1)      =   f(n)
0  c        c^(n-1)         c^n


----------------------------------

设方阵 A, B， 已知C，验证 AxB == C

对于小数据暴力即可
对于大数据可用一位行向量α*A*B，通过结合律降维
    α*A*B == α*C
α取随机值或者类似于1,2^2,3^2....等序列减少碰撞几率
